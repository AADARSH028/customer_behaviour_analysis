## 📌 Project Overview  
This project focuses on analyzing customer purchasing behavior using a retail/e-commerce dataset.  
The goal is to extract insights that help businesses improve revenue, customer retention, and marketing strategies.

---

## 🎯 Business Problem  

- Low customer retention  
- Ineffective marketing campaigns  
- Poor understanding of product demand  
- Need for customer segmentation  

---

## 📂 Dataset Information  

- **Type:** Retail / E-commerce  
- **Level:** Customer Transaction Data  
- **Records:** 5000+ customers  

### 🔢 Features
- Numerical → Age, Purchase Amount, Previous Purchases  
- Categorical → Category, Payment Method, Season  

---

## 🧹 Data Cleaning & Preprocessing  

✔️ Handled missing values using business logic  
✔️ Fixed category inconsistencies  
✔️ Removed duplicates  
✔️ Standardized column names  
✔️ Exported clean data to SQL Server  

---

## 🧠 SQL Analysis  

Key questions solved:

- Which category generates the highest revenue?  
- Do discounts increase purchase value?  
- Revenue by gender  
- Top & bottom rated products  
- Shipping impact on spending  
- Subscription vs non-subscription analysis  
- Customer segmentation  

---

## 📈 Key Insights  

### 💰 Revenue Insights  
- Electronics generate the highest revenue  
- Outerwear performs the lowest  

### 🎯 Discount Strategy  
- Discounts increase average purchase value  

### 👥 Customer Segmentation  
- Loyal Customers → 3085  
- Returning Customers → 1354  
- New Customers → 561  

### 🚚 Shipping Analysis  
- Express shipping leads to higher spending  

### 💳 Subscription Insights  
- Subscribers spend significantly more
